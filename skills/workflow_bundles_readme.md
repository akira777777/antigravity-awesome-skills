# Workflow Bundles

This file is a lightweight pointer for people browsing the `skills/` directory.
The canonical workflow and bundle documentation now lives outside this folder:

- [`../docs/users/bundles.md`](../docs/users/bundles.md) for curated starter packs
- [`../docs/users/workflows.md`](../docs/users/workflows.md) for step-by-step execution playbooks

Use this file as a quick reminder that workflow bundles are implemented as normal
skills in this directory, such as:

- `development`
- `ai-ml`
- `cloud-devops`
- `testing-qa`
- `react-nextjs-development`
- `python-fastapi-development`

If you are updating workflow-bundle documentation, edit the docs in `docs/users/`
first and only keep short cross-links here.
